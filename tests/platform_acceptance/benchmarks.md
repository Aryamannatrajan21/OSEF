# Canonical External Benchmark Suite

For Milestone A (Platform Validation), OSEF will benchmark against these canonical open-source repositories to validate that the Engineering Knowledge Graph architecture scales without issue.

## Python
- **FastAPI**: Validate robust API parsing, decorators, models.
- **Django**: Validate large-scale monolithic architectures.
- **Typer**: Validate CLI parsing and dependencies.

## Infrastructure
- **Kubernetes**: (Helm charts, manifests).
- **Docker Compose examples**: Complex multi-service orchestration.

## JavaScript / TypeScript
- **React**: Component architectures.
- **Next.js**: Server-side rendering, routing.

## Go
- **Cobra**: CLI commands.
- **Gin**: API frameworks.

## Rust
- **Tokio**: Concurrency and async/await traits.

## Java
- **Spring Boot PetClinic**: Enterprise patterns, dependency injection.
