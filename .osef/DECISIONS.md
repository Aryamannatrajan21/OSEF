# OSEF Major Decisions

Permanent log of decisions impacting continuity.

1. **Python as core implementation language:** For native AI ecosystem integration.
2. **AST over regex:** For semantic accuracy.
3. **EKG as single source of truth:** Overcoming tree traversal limitations.
4. **EPE Declarative Rules:** Extracting imperative logic from analyzers.
5. **EPSDK Extension Host:** Sandboxing all plugin interactions for safety.
6. **Apache License 2.0:** Selected to encourage enterprise adoption and plugin ecosystem growth over MIT.
