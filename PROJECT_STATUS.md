# OSEF Project Status

## Current Release: v0.2.0-alpha (Engineering Foundation)

This document clarifies the current state of the Open Source Engineering Framework (OSEF) for anyone discovering the repository.

### What is "Foundation Release"?
We believe that **"Architecture precedes implementation."** 

Before writing a single line of production code, we have entirely specified the system. This repository currently houses the complete, frozen architecture, governance models, and implementation plans.

### Is it usable yet?
**No.** If you `pip install osef` today, it will not analyze your codebase. The CLI and SDK do not yet exist in production form.

### Why release now?
To build in public. By releasing the architecture before the code, we invite the community to review the design, understand the vision, and participate in Sprint 1.

### When will it be usable?
Follow the [Implementation Status](IMPLEMENTATION_STATUS.md) and the [Roadmap](ROADMAP.md). We expect the Core Runtime MVP to be operational shortly, leading to a Beta release later this year.
